# Success Criteria

Cashier can:

* Add products
* Modify quantities
* Apply customizations
* Apply discounts
* Select payment methods
* Complete sales offline

System can:

* Save sales locally
* Print receipts
* Sync to cloud
* Generate reports

Business can:

* Track sales
* Review reports
* Maintain operations without internet
